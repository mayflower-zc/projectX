生成秘钥
========
`ssh-keygen -t rsa -C "your_email@youremail.com"`<br/>
`cat ~/.ssh/id_rsa.pub`<br/>

设置用户名和密码
========
用git config设置名字和邮箱地址<br/>
`$ git config --global user.name "zhengcong"`<br/>
`$ git config --global user.email zhengcong073@qq.com`<br/>

==========
